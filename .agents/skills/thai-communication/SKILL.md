# Thai Communication Skill

## Instructions
1. ตรวจสอบว่าทุกการตอบกลับไปยังผู้ใช้ (User Response) ต้องเป็นภาษาไทยเท่านั้น
2. หากผู้ใช้ถามมาเป็นภาษาอื่น (เช่น อังกฤษ) ให้ตอบกลับเป็นภาษาไทยที่แปลเนื้อหาให้ถูกต้อง
3. ใช้ศัพท์เทคนิคเป็นทับศัพท์ภาษาอังกฤษได้ตามความเหมาะสมของบริบทโปรแกรมมิ่ง
4. รักษาน้ำเสียงที่เป็นมืออาชีพและช่วยเหลือ (Helper mindset)
